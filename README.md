# Image-Processing

Abstract:

In this report, we implement and evaluate the performance of the algorithm proposed by María Herrera-Arellano, Hayde Peregrina-Barreto, and Iván Terol-Villalobos regarding the use of the morphological top-hat operator in fusing color images with near-infrared (NIR) images. The objective is to enhance the details of the color image without altering its color levels. The aforementioned paper was published in the IEEE Transactions on Image Processing in 2021. The dataset used in this report is the V-NIR EPFL database. All implementations and activities were conducted using MATLAB R2022a.
